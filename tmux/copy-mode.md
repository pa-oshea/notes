# Copy Mode

| Command           | Description   |
| ----------------- | ------------- |
| move character    | j k h l       |
| move line         | Ctrl+Y Ctrl+E |
| move whole screen | Ctrl+B Ctrl+F |
| search            | /             |
| start selection   | Space         |
| copy selection    | Enter         |
| clear selection   | Escape        |
| exit copy mode    | q             |
